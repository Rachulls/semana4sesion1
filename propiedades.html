PROPIDADES DE FUENTES

    'font-family'

    Define la familia de fuentes que se aplicará al texto.

    body { font-family: "Helvetica Neue", Arial, sans-serif; }

    'font-size

    Establece el tamaño de la fuente. Puede ser especificado en píxeles, em, rem, porcentajes.

    h1 { font-size: 24px; }

    'font-weight'

    Define el grosor de la fuente. Puede tomar valores como normal, bold, bolder, lighter

    p { font-weight: bold; }

    'font-style'

    Define el estilo de la fuente, como normal, italic o oblique.

    em { font-style: italic; }
